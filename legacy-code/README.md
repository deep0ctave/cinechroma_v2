# Legacy Code Archive

This folder contains the complete history of earlier development iterations of the CineChroma project. All commit history from the original repositories has been preserved.

## Archived Repositories

### 1. cine_chroma
- **Original Repository**: https://github.com/deep0ctave/cine_chroma
- **Description**: Initial attempt at building the CineChroma application

### 2. cinechroma
- **Original Repository**: https://github.com/deep0ctave/cinechroma
- **Description**: Second iteration of the CineChroma project

### 3. cinechroma_legacy
- **Original Repository**: https://github.com/deep0ctave/cinechroma_legacy
- **Description**: Legacy version of the CineChroma application

## Note

These repositories have been merged into the main cinechroma_v2 repository using git subtree to preserve all commit history. The original repositories can now be safely deleted from GitHub if desired, as their complete history is preserved here.

All commits from these repositories are now part of the main repository's git history and can be explored using standard git commands.
